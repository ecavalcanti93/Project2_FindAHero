<p id="description">Imagine a world where, with just one click, a hero can come to your aid when you need it most. **Find A Hero** is the platform that connects heroes with civilians in danger. Here, any hero can sign up to offer help and ensure everyone's safety. Whether it's an unexpected rescue or an imminent emergency, just open the app and find the right hero to save the day! 💥</p>

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository</p>

```
git clone repo_url
```

<p>2. Install npm modules</p>

```
npm install
```

<p>3. Start development server using below command</p>

```
npm run dev
```
